# OpenCode / Continue.dev 等（OpenAI 兼容）

若工具支持 **OpenAI 兼容 Base URL**（名称因产品而异）：

| 变量 | 值 |
|------|-----|
| API Key | `ptd_...` |
| Base URL | `https://YOUR_ORIGIN/api/v1/openai/v1` |
| model | `api-usage-guide` 中的 `provider/model` |

以各产品官方文档为准；Continue 等常在 VS Code 设置里提供 **OpenAI-compatible** 或 **自定义 API** 项。
