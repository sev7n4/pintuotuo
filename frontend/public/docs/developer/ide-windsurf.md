# Windsurf（Codeium Cascade）

在 **Cascade / AI** 或 **Settings → Advanced** 中查找：

- **Custom OpenAI endpoint** / **OpenAI Base URL** → `https://YOUR_ORIGIN/api/v1/openai/v1`  
- **API Key** → `ptd_...`  
- **Model** → `api-usage-guide` 中的 `provider/model`  

若仅支持 Codeium 内置模型而无法填自定义 Base URL，则无法直连拼脱脱；需以产品当前版本说明为准。
