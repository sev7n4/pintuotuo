```
数据流详解:
┌───────────────────────────────────────────────────────────────────┐
│                                                                   │
│  用户请求:                                                        │
│    POST /openai/v1/chat/completions                               │
│    { "model": "gpt-4o", "messages": [...] }                       │
│                                                                   │
│  Backend处理:                                                     │
│    1. 认证 → 预扣费 → 路由决策 → 选择API Key                      │
│    2. route_mode = litellm → 构建user_config                      │
│    3. 转发到: https://litellm-overseas.yourdomain.com/v1          │
│       {                                                           │
│         "model": "proxy-model",                                   │
│         "user_config": {                                          │
│           "model_list": [{                                        │
│             "model_name": "proxy-model",                          │
│             "litellm_params": {                                   │
│               "model": "openai/gpt-4o",                           │
│               "api_key": "sk-xxx",                                │
│               "api_base": "https://api.openai.com/v1"             │
│             }                                                     │
│           }]                                                      │
│         }                                                         │
│       }                                                           │
│                                                                   │
│  LiteLLM处理:                                                     │
│    1. 验证 LITELLM_MASTER_KEY                                     │
│    2. 解析 user_config → 获取真实model + api_key + api_base       │
│    3. 转发到 https://api.openai.com/v1/chat/completions           │
│    4. 返回响应给Backend                                           │
│                                                                   │
│  Backend返回:                                                     │
│    结算实际用量 → 返回响应给用户                                    │
│                                                                   │
└───────────────────────────────────────────────────────────────────┘
```
