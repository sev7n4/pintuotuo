## 2. 浏览器走查清单（请人工勾选）

- 新注册 **普通用户**（买家路径），未误入商户后台  
- 首页快捷入口（热销/拼团/秒杀等）→ 卖场筛选参数正确保留  
- 商品详情选 SKU → 加购 → 购物车金额与数量正确  
- **单商品** 结算 → 创建订单 → 跳转支付页  
- **多商品** 结算 → 提示「多笔订单」→ 订单列表可逐笔支付  
- 支付成功/失败后订单列表与详情状态一致  
- 头像下拉 → 我的服务 → Token / 消费明细 / 订单链路连贯  
- 商品详情「用户评价」Tab：与当前产品能力一致（占位或已接 API）  
- 手机宽度（或 DevTools 移动设备）：首页、卖场、结算、支付可用
