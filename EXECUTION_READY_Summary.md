### 🎯 What Gets Built (8-Week MVP)

#### C-End User Features
1. User registration and authentication
2. Browse AI Token products with filters
3. View product details and active groups
4. Create or join group for social buying
5. Complete purchases with payment
6. View order history and status
7. Use tokens/API with real-time tracking
8. Referral rewards and social sharing

#### B-End Merchant Features
1. Merchant account registration and verification
2. Upload and manage API Keys securely
3. Create and manage Token SKUs
4. Set pricing (solo vs. group)
5. View real-time sales dashboard
6. Manage group mechanics (subsidy, auto-complete)
7. Access settlement and payout information
8. Analytics and performance metrics

#### Platform Features
1. Payment processing integration
2. Automatic group completion algorithm
3. Real-time group formation and countdown
4. API Key hosting and secure storage
5. Token consumption tracking and billing
6. Merchant commission calculation
7. User subsidy and referral reward system
8. Monitoring, logging, and alerting
